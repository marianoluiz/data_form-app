# Personal Data Forms Website

### Tech stacks:

###### Frontend:

- React - frontend js library
- Scss - css pre-processor
- Bootstrap - frontend css framework
- Vite - frontend build tool

###### Backend:

- Hibernate / HQL - orm
- Java - backend
- Apache Tomcat - server

*on going

###### Database:
- MySQL 

## Steps to run Frontend:

- `cd frontend`
- `npm install`
- `npm run dev`
